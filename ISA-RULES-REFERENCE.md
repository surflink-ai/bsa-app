# ISA/WSL Rules Reference — Extracted from Official Documents

## Priority System (ISA Rulebook April 2025)

### Display
- Priority Judge uses **colored display system corresponding to jersey colors**
- Vertical = top to bottom, Horizontal = left to right
- **Surfer's responsibility to check priority at all times**

### Priority Lost When:
- Surfer rides a wave
- Makes committed paddle and misses (hands left rails or attempted to stand)
- Paddles outside Primary Take-off Zone

### Priority NOT Lost When (3+ surfer heats):
- Paddling alongside higher-priority surfer who catches the wave
- Blocked by higher-priority surfer

### 4-Surfer Heat Priority (BSA standard):
1. First surfer rides → gets 4th priority
2. Remaining 3 have priority over 1st, can paddle freely
3. Second surfer rides → remaining 2 still have priority
4. Third surfer rides → initial order established:
   - Surfer yet to catch wave = 1st priority
   - Others ordered by return to Primary Take-off Zone
5. 1st > all, 2nd > 3rd+4th, 3rd > 4th only

### Priority Suspension:
- If priority surfer paddles OUTSIDE Primary Take-off Zone → priority suspended
- Can be reinstated when they return
- Also suspended on equipment damage while getting replacement

### Interference + Priority:
- Interference = lose priority, Priority Judge determines new position
- Blocking rule: priority surfer loses priority if deliberately blocking

## Interference Penalties

### Penalty 1 (Non-priority interference):
- Called by majority of judging panel
- Hindering scoring potential of surfer with right of way

### Penalty 2 (Priority interference):
- Called when surfer without priority interferes with priority surfer
- Can be called by Head Judge alone if majority can't see

### Penalty Application:
- **Second highest scoring wave** is halved (50%)
- If only one wave scored, that wave is halved
- If no waves scored, first wave scored gets halved

### Double Interference:
- Two interferences in same heat = disqualification (last place)

## Scoring/Tabulation (ISA Judging Manual)

### Scale: 0.0 – 10.0 (one-tenth increments)
- 0.1–1.9: Bad
- 2.0–3.9: Poor  
- 4.0–5.9: Average
- 6.0–7.9: Good
- 8.0–10.0: Excellent

### Judge Panel:
- 5 judges standard, drop highest + lowest, average remaining 3
- 3 judges minimum: straight average
- Head Judge oversees, can provide average score if judge misses a wave ("M" notation)

### Best 2 Waves Count (since late 1990s)

### Scoring Procedure:
1. Start with whole + half numbers (0.5 increments)
2. 2nd scale: use .2, .3, .7, .8 once whole/half are established
3. 3rd scale: use .1, .4, .6, .9 only when 2nd scale is established
4. Compare every wave back to: first wave, current highest, day's highest

### Judging Criteria (ISA 2010 – Current):
"A surfer must perform radical controlled manoeuvres in the critical sections of a wave with speed, power and flow. Innovative/progressive surfing as well as the combination and variety of repertoire (of major manoeuvres) will be taken into consideration when rewarding points for a surfers performance."

### Key Criteria Elements:
1. Radical Controlled Manoeuvres
2. Critical Sections (pocket, closest to curl)
3. Speed, Power and Flow
4. Innovative/Progressive
5. Combination and Variety of Repertoire
6. Degree of Difficulty & Commitment

### If Judge Misses Part of Wave:
- Write "M" in scoring box
- Head Judge provides average from other judges' scores
- Do NOT copy or guess
